# DataScience_project_2 - Fraud Detection Pipeline

---

## 📌 Project Overview

This project demonstrates a complete **Machine Learning Classification Pipeline** to predict **Order Status** using an E-commerce dataset.

The project covers data preprocessing, feature engineering, class balancing using **SMOTE**, model training, hyperparameter tuning, and performance evaluation.

---

## 🎯 Project Objectives

✅ Predict Order Status using Machine Learning

✅ Handle Imbalanced Data using SMOTE

✅ Compare Multiple Classification Models

✅ Evaluate Model Performance using different metrics

---

## 📂 Dataset

The dataset contains E-commerce order information including:

- 📦 Product
- 🔢 Quantity
- 💰 Unit Price
- 💳 Payment Method
- 🎟 Coupon Used
- 🌐 Referral Source
- 🛒 Items in Cart
- 📅 Order Month
- 📅 Order Day
- 💵 Total Price
- 📊 Average Item Price

**Target Variable:**
- ✅ Order Status
